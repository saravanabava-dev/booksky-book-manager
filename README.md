# 📚 Booksky Project

A simple and creative web application to manage and showcase books.  
Users can add books, view them, and delete them dynamically.

## 🚀 Live Demo
https://lnkd.in/gmdYFjhr

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## ✨ Features
- ➕ Add new books
- 📖 Display book details
- ❌ Delete books
- 🎨 Clean and simple UI

## 📂 Project Structure

Booksky/
│── index.html        # Main webpage
│── style.css         # Styling
│── script.js         # Functionality


## ⚙️ How It Works
- User clicks **Add Book**
- A popup form appears
- Enter book details
- Book gets added to the list
- Can remove any book using delete option

## 📌 How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
